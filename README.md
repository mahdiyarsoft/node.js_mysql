# node.js_mysql
connect mysql with node.js (Js)

## Usage
> 1- first step install node.js on your system.
> 
> 2- `npm init -y`
> 
> 3- `npm install -g nodemon`
> 
> 4- `npm install --save-dev nodemon`
> 
> 5- `npm install --save-dev mysql`
> 
> 6- `touch app.js`
> 
> 7- to run your project `nodemon app.js`
